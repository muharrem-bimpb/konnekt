Support & Hilfe
❓ Häufige Fragen
Event erscheint nicht auf der Karte?
Stelle sicher dass du eine Stadt oder Adresse eingibst — Events ohne Standort werden zwar gespeichert, aber nicht auf der Karte angezeigt.
Punkte nicht gutgeschrieben?
Punkte werden sofort nach erfolgreicher Aktion gutgeschrieben. Lade die Seite einmal neu. Bei anhaltenden Problemen kontaktiere uns.
Coupon Code zeigt Fehler beim Einlösen?
Zeige dem Geschäft den QR-Code / Text-Code direkt auf deinem Bildschirm. Der Code ist einmalig gültig.
Anmeldung funktioniert nicht?
Versuche "Mit Google anmelden" als Alternative. Bei Passwort vergessen nutze den Magic Link Button (kein Passwort nötig).
